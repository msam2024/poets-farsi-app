# شاعران و اشعار (PWA فارسی)

یک وب‌اپلیکیشن بسیار ساده و سریع برای نگه‌داری **فهرست شاعران** و **اشعار** به زبان فارسی.
روی آیفون به‌راحتی قابل استفاده است و می‌توانید آن را به **Home Screen** اضافه کنید تا مثل اپ اجرا شود (آفلاین نیز کار می‌کند).

## امکانات
- راست‌به‌چپ (RTL) و سازگار با فونت‌های فارسی
- جستجو در نام شاعر و عنوان/متن شعر (با نرمال‌سازی نویسه‌های عربی/فارسی)
- ذخیرهٔ داده‌ها در مرورگر (localStorage)
- PWA + Service Worker برای اجرا بدون اینترنت

## نصب و اجرا
1. همه فایل‌ها را روی هر هاستی بارگذاری کنید (مثلاً GitHub Pages، Netlify، Vercel).
2. روی آیفون در Safari باز کنید.
3. از منوی Share، گزینهٔ **Add to Home Screen** را بزنید.

## نکات
- اگر داده‌ها زیاد شود، می‌توانید به IndexedDB مهاجرت کنید.
- برای فونت اختصاصی، می‌توانید فونت «Noto Sans Arabic» را اضافه کنید و در `styles.css` آن را تنظیم نمایید.
- برای انتقال داده بین دستگاه‌ها، می‌توانیم قابلیت **Export/Import** JSON اضافه کنیم.
